# Auto Document Scanner (OpenCV, No Deep Learning)

A fully automatic document scanner using OpenCV without deep learning. 
It detects document corners, applies perspective transform, and preserves natural colors.

## Credits

This project is based on the OpenCV Document Scanner by Andrew Campbell:
https://github.com/andrewdcampbell/OpenCV-Document-Scanner

Modifications:
- Removed manual corner selection
- Removed whitening/thresholding
- Fully automated pipeline
- Improved performance under different lighting conditions
